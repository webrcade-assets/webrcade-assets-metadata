These educational games with the Mr. Men characters from Roger Hargreaves' children books are for kids aged 5 to 8 years. It features these fun games that also develop reading skills: Mr Noisy's Word Game - Learn about opposites, comparatives and superlatives. There are nine different games to play, each of which can be selected from the main menu. Other games allow various combinations of reading and writing exercises ranging from matching up a picture with the correct word to choosing the opposite to a displayed word. Read with Mr Bounce - Learn words like 'above', 'inside', and 'on'. Again, there are several different options to teach different uses of the words. Included with the game is a specially designed keyboard overlay with clear 'small' letters cutting out the confusing clutter on a computer keyboard and an illustrated instruction book to help an adult introduce the program to children, plus separate notes on the programs and the educational aspects.
