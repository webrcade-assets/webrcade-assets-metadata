Rainy Day Fun is a collection of games and activities for young children. Contains several short games: Banner Maker: Create banners to celebrate events. Coloring Book: Color a picture. Dot to Dot: Connect the dots to make a picture. Fortune Teller: The computer tries to answer questions posed about the future/ Invitation Maker: Create invitation cards for Birthdays, Halloween, or Valentines. Mask Maker: Create a mask to color and wear. Maze Maker: Create a maze and watch a mouse run through it. Paper Airplane: Create a paper airplane Pin the Tail on the Donkey: Try to pin the tail on a moving donkey. Sliding Puzzle: A 4 x 4 sliding block puzzle
