Vic Cricket is a 1-player sports strategy game for multiple systems. The program simulates a cricket game versus the computer The player can set the strength of each side and the number of overs. Individual team members can be named, and the type of game can be set. Numerous tactical choices including: run rate, order, protecting the tail, or setting the field can be changed. During the game, the player sets which bowler will step up, and the bowler's performance is automatically displayed on the screen.
