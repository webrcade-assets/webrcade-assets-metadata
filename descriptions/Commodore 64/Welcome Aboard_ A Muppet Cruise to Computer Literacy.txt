Welcome Aboard is an educational game that tries to help people become more computer literate with the help of the Muppets. There are 5 different activities included: The Bridge: help Kermit the Frog to steer the ship using a Logo-like programming language The Game Room: play an arcade style game under the protests of Sam the Eagle The Joke Library: learn fundamentals of database management by organizing Fozzie Bear's joke database The Message Center: learn how to use word processing and email with Scooter The Salon de Beaute: use a computer to find the perfect hairdo and outfit for Miss Piggy
