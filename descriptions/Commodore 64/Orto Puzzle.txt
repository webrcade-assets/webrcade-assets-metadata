Orto Puzzle is an educational game for kids. The player's task is to choose the correct spelling. The correct spelling needs to be selected from two different letters. The game highlights a part of the word the player needs to decide on. If correct, the word is replaced by a part of the picture, if not, it remains on the board and deduct one life. All twelve words must be solved with up to three errors. The game features built in spelling rules and allows to choose specified orthographic set of words.
