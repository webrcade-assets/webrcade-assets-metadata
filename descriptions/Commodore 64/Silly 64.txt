Silly 64 contains several bizarre programs for the Commodore 64. Annoying Repetitious Message: Repeatedly displays the users message on the screen, filling the entire page. Reality: Leaves the program Silly Crisis: The Commodore 64 begins to regret its life decisions. Silly Data: The program creates a number of different graphs to display completely frivolous data. Silly Poem: The program takes the player's choices to a number of questions and creates a number of randomized poems.
