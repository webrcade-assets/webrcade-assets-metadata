Bop is an educational game that teaches the player Chisanbop, a counting method developed in Korea in the 1940s. It allows the player to count to 99 using two hands. The game has four modes: A demonstration mode that slowly counts from 1 to 99, showing all values A look-up mode where the player can give a number and see how the hand should be held An exercise mode where the player must select the fingers to show to correspond to a given number An exercise mode where the player must guess the number the hand currently shows
