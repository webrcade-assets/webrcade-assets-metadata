Wu Lung is an adaption of the tile game Mahjong solitaire which revolves around removing tiles which are stacked in five levels. To remove a tile, it has to be matched with another one with the same motive. This version features a tournament mode for two players, the option to view the plains individually and a function to undo mistakes. Players can also search for a certain tile which in turn gets highlighted. Another feature are different colours for every level which can be changed by the player.
