La Culture Générale Pour les Nuls returns today on Nintendo DS with the 4000 questions from the 1st edition updated, and more than 2,500 unpublished questions on new themes: Famous quotes, musical quiz, ancient Egypt ...

 This 2nd edition is accompanied by new features: "Epelle" allows for example to write the answers directly through a virtual keyboard, the mental computing event has been redesigned to add fun to the test and the famous HOUSSED game  has been integrated.
