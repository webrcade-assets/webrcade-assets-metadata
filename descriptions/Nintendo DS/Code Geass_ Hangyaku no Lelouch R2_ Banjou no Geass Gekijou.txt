Code Geass: Lelouch of the Rebellion R2 - Banjou no Geass Gekijou is the 3rd game based on the Code Geass franchise, developed and published for the Nintendo DS by Bandai Namco Games and released on Aug 7, 2008 in Japan. There has been no international release. It is a collection of minigames featuring Super Deformed forms of the characters. The player moves along a board through dice rolls, landing on different spots to activate minigames. The minigames are parody-style events with multiple genres. These include helping Jeremiah grow oranges, racing against C.C. and Shirley in swimming, and a sidescrolling beat-em-up featuring Kallen in Guren-like armor.
