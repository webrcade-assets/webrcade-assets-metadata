Sangokushi DS is a remake of Romance of the Three Kingdoms III: Dragon of Destiny (Sangokushi III). It has 2 main game mode: Orthodox Mode which is similar to the original RTK III, and DS Mode which makes many modifications to the game system. The game includes not only one new scenario, but also several challenge scenarios which can be cleared by reaching given targets instead of unifying the whole China. It also has new officers, new officer abilities, new items and a local multiplayer mode which allows a player to wage single combats with another one.
