Crates is a single-player, shareware, logic puzzle game similar to Chip's Challenge.The game consists of a series of rooms in which are a number of crates. The player's objective is to manoeuvre these crates into the designated storage area in the minimum possible number of moves. The shareware version has five rooms while the full registered version has fifty, ten rooms on five floors, and a room editor. Each room has a par number, the number of moves with which the player should be able complete the puzzle, some of these numbers can be beaten.
