Amado is a single-player, shareware, keyboard-controlled puzzle game. The game is played on a puzzle grid in which each of the cells is coloured red, blue or yellow. Using the keyboard the player moves from one cell to the next and as they do so the colour of the cell they are moving to changes colour according to a set of rules, the object of the game being to replicate a given pattern. The game has no sound other than a muted blup as the player moves from cell to cell, it has three grid sizes, 4x4, 5x5 and 6x6.
