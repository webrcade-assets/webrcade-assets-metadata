3D Game Alchemy is a compilation of mods for DOOM, DOOM II, Heretic and Hexen: Beyond Heretic. The levels can be selected and filtered with the DoomShell interface in an adapted version for this compilation. It can also be used to load levels from outside the package.
In addition, the package includes a printed book with the same name (800 pages) which teaches how to make user levels for id Tech 1, a .pdf supplement (56 pages) specifically about Hexen modding, a graphics library and two map editors: WADED (DOS, freeware) and WadAuthor (Windows 3.x, fully registered shareware version).
