10 Tons Of Games: Mega Collection 1 is a compilation of shareware games written for both DOS and Windows.Games are grouped into categories and are accessed using the RomCat navigation / menu system which also loads, installs and in some cases executes the games. The game categories on this CD are divided into two main groups which are broadly DOS games and Windows games, though there is some crossover.Within each of these main groups the games are grouped together by type such as Adventure games, Puzzle games, and Board games. Some groupings such as Card games and Arcade games are so large that they are further sub-divided.
