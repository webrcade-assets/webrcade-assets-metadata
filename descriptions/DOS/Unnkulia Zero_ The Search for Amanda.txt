Unnkulia Zero: The Search for Amanda is a text adventure game with many of the locations in an underground caverns, developed with TADS - Text Adventure Development System. Corresponding with the title, the game is set before the other Unnkulian games, allowing to learn the history of the Valley. As the Valley King's most trusted warrior, the player must rescue Lady Armada from the evil Unnkulian demons. The game features extensive descriptions of the locations. The game is fully keyboard controlled by using dozen types of commands available. Selected locals of Unnkulia and the Valley will converse and interact with the player.
