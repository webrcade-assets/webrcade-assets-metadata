Unnkulian Underworld : The Unknown Unventure is a text adventure game that started the satirical Unnkulian saga, developed with TADS - Text Adventure Development System. As Kuulest's slave the player must recover the Orb of Studosity from the
evil Unnkulians. The adventure starts when the Hermit Kuulest dies, leaving a cryptic message about saving the world. The game features extensive descriptions of the locations. The game is fully keyboard-controlled by using a dozen types of commands available.
