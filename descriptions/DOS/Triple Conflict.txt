Triple Conflict is a compilation which contains three games:

The fantasy RTS game WarCraft: Orcs & Humans
The colonial style turn-based strategy game Conquest of the New World: Deluxe Edition
The sci-fi RTS game M.A.X.: Mechanized Assault & Exploration.

All games are on a single CD each, while the fourth CD in the package contains all the manuals, as Adobe PDF files.
