CosmoServe is a single player text adventure about the trials of a software developer. The game starts on Friday September 7th 2001 and the player, R. J. Wright a self employed computer consultant and plumber, sits down to finish a software package for a client. A courier will arrive at eight in the morning to pick it up.Needless to say things do NOT go according to plan.CosmoServe is a game based on the work of a software developer so a lot of things happen on the player's in-game computer which accepts some DOS commands, play a game within a game and more. There are basic graphics at points within the game and bleeping sound effects for the in-game computer.
