Apollo Collection 2 is a set of sixteen aircraft in an add-on package, requiring the 3rd-party application BAO FlightShop, to use with Microsoft Flight Simulator (v5.0). While the original game is DOS only, this add-on includes Windows 3.x and Windows 9x installers next to a manual DOS operation to copy the content.
