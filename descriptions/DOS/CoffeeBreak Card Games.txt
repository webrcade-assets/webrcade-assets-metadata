CoffeeBreak Card Games is a collection of card games: the original release had 52 Pickup, 31, Solitaire (Klondike), Crazy 8's, Matching Game, and Blackjack. This was later expanded to include three more variants of solitaire ( Kings' Thrones, Streets and Allies, and Golf). Options include different card backs, different colors, and sound/music. All games can be played with the mouse or keyboard.This game was distributed as Shareware. The original shareware version only includes 52 Pickup and Crazy 8's. Later releases also include Solitaire (Klondike), Matching Game, and Blackjack.The game was renamed from CoffeeBreak Card Games to Cavalcade O' Card Games due to an existing game using a similar name.
