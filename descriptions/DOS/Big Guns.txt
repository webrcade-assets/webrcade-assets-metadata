Big Guns is a battleship duel game where two Iowa-class battleships fights.One player controls USS New Jersey and the other player (or computer) controls USS Iowa. Each battleship carries 64 planes of which 3 at a time can be used to attack the other battleship. The game also features USS Harry E. Yarnell, USS Biddle, USS Richmond K. Turner, USS Josephus Daniels, Titanic, Olympic and Lusitania. The three civilian ships fills the function of depot ships and will supply the combatants with fuel and new planes. In addition to the main guns and planes each battleship also has 99 Harpoon missiles that can be used to sink the enemy. The number of kills to win the game can be set between 1 and 20.
