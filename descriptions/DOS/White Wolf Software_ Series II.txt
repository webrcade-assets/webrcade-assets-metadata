White Wolf Software: Series II is one of a series of CDs published by Wiz Technology on their White Wolf label. This CD contains the following shareware products:* A: RaptorB: JetpackC: EGA Trek: The Mongol InvasionD: Stellar Explorer from Dynamic Learning SystemsE: Cheat: a list of game cheat codesF: Address Librarian - a database for addressesG: Movies TriviaH: Basketball Stats Power: a database for basketball statsI: ExpressCheck: a home finance programJ: Flying TigersThere is no 'Install All' option, the user can only install one product at a time. After installation the user is kicked out into DOS in the home directory of the installed software.
