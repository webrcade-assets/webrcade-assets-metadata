Tommy's Patience II is a shareware, single-player card game.This implementation includes four games:* Betsy Ross, aka Quadruple AllianceCorner Card, aka Four SeasonsPyramidSir Tommy, aka Old Patience or Try AgainThe game is entirely keyboard controlled. It has optional basic sound effects, a demo mode, and command line switches to start the game with sound turned off and to force either a colour or monochrome display.
