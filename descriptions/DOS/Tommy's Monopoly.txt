Tommy's Monopoly is an implementation of the classic board game for up to eight players, in the single player game the computer takes the role of an opponent.The game has all the standard features that would be expected, automatic save state on exit, a short game mode as well as a super short game mode, and a time limit function. The game also has a speed control which allows the player(s) to adjust the time the on-screen messages are displayed thus potentially making the game faster. Unusually it also has a demo mode and a pitch control that adjusts the frequency of the sound the dice make when rolling.The game has command line switches to play in monochrome or colour mode and to start the game without sound.
