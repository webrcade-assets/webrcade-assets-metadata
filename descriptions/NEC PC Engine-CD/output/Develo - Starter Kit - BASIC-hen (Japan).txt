Develo Starter Kit: Basic-hen is a programming tool that allows users to create software using the Basic language. Also included on the disc are three games made with the tool: Sky is a simple arcade game where the player has to prevent a ball from falling through holes in the ground; G-Tennis takes place in a pseudo-3D tube and involves rotating a paddle to touch incoming balls; in Meiro Game the player scrolls the screen to find a way through a procedurally generated labyrinth.
